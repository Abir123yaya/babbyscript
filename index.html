babbyscript reference
every word you need to know
storing values
remember name is "Alice"
remember age is 25
remember happy is yes
— use yes / no for true / false
showing things
say "hello!"
say name
say "you are" age "years old"
— mix text and variables freely
asking questions
ask "what is your name?" into name
say "hello," name

— a little popup asks the question
— whatever they type gets stored "into" your word
— numbers typed in (like 7) work as real numbers
making decisions
if age is more than 18
  say "adult"
otherwise
  say "kid"
done

conditions you can use:
x is y        (equal)
x is not y   (not equal)
x is more than y
x is less than y
x and y      (both true)
x or y       (either true)
repeating things
repeat 5 times
  say "hi!"
done

for each fruit in apples oranges bananas
  say fruit
done
teaching new tricks
teach greet person
  say "hello," person
done

do greet "Alice"
do greet "Bob"

— teach gives it a name
— do runs it

remember half is total / 2
